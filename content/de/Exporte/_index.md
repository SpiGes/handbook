Sie loggen sich als Nutzer ein und gelangen auf die Startseite (1).

### Startbildschirm

{{<insertImage image="exp1_d.png" class="edge max-w-90">}}

&nbsp;

Sie klicken auf das Tabellenblatt Datenexport (2) und wählen die gewünschten Datenjahre aus (3).

### Tabellenblatt Datenxport

{{<insertImage image="exp2_d.png" class="edge max-w-90">}}

&nbsp;

Sie gelangen auf eine Übersicht, wo Sie die Granularität der zu exportierenden Daten auswählen (4); entweder wählen Sie das Schweizweite Datenset oder Daten auf Unternehmens- oder Standortebene. Je nach Ihren Downloadrechten lassen sich diese Ebenen aktivieren.

### Übersicht der zu exportierenden Daten

{{<insertImage image="exp3_d.png" class="edge max-w-90">}}

&nbsp;

Nach der Wahl der Granularität gelangen Sie zu einer Ansicht, wo Typ oder Format der KVG Daten ausgewählt werden muss. Wenn Sie auf die Zeile des entsprechenden Datentyps scrollen, können Sie rechts auf den Downloadbutton klicken (5).

### Typ oder Format der KVG Daten

{{<insertImage image="exp4_d.png" class="edge max-w-90">}}

&nbsp;

Wenn Sie den Downloadbutton klicken poppt eine Nachricht auf, dass Sie eine Downloadlink per Mail erhalten (6).

### Download Button

{{<insertImage image="exp5_d.png" class="edge max-w-90">}}

&nbsp;

Nachdem Sie das Mail erhalten haben, können Sie dort einen Link aktivieren (7).

### Bestätigung per Email

{{<insertImage image="exp6_d.png" class="edge max-w-90">}}

&nbsp;

Sie werden auf die Downloadseite der SpiGes Plattform gleitet und die KVG Daten werden im Download Ordner Ihres Computers gespeichert.
