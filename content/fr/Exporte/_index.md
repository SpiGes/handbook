Vous vous connectez en tant qu'utilisateur et accédez à la page d'accueil (1).

### Écran de démarrage

{{<insertImage image="exp1_f.png" class="edge max-w-90">}}

&nbsp;

Vous cliquez sur l'onglet Exportation des données (2) et sélectionnez les années souhaitées (3).

### Feuille de calcul exportation des données

{{<insertImage image="exp2_f.png" class="edge max-w-90">}}

&nbsp;

Vous accédez à un aperçu où vous pouvez sélectionner la granularité des données à exporter (4) ; vous pouvez choisir soit l'ensemble de données pour toute la Suisse, soit les données au niveau de l'entreprise ou du site. Selon vos droits de téléchargement, ces niveaux peuvent être activés.

### Aperçu des données à exporter

{{<insertImage image="exp3_f.png" class="edge max-w-90">}}

&nbsp;

Après avoir sélectionné la granularité, vous accédez à une vue dans laquelle vous devez sélectionner le type ou le format des données LAMal. Lorsque vous faites défiler la ligne du type de données correspondant, vous pouvez cliquer sur le bouton de téléchargement à droite (5).

### Type ou format des données LAMal

{{<insertImage image="exp4_f.png" class="edge max-w-90">}}

&nbsp;

Lorsque vous cliquez sur le bouton de téléchargement, un message s'affiche pour vous informer que vous recevrez un lien de téléchargement par e-mail (6).

### Download Button

{{<insertImage image="exp5_f.png" class="edge max-w-90">}}

&nbsp;

Après avoir reçu l'e-mail, vous pouvez activer le lien qui s'y trouve (7).

### Confirmation par e-mail

{{<insertImage image="exp6_f.png" class="edge max-w-90">}}

&nbsp;

Vous serez redirigé vers la page de téléchargement de la plateforme SpiGes et les données LAMal seront enregistrées dans le dossier de téléchargement de votre ordinateur.
